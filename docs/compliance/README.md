# Compliance Pack

This folder contains the repo-side governance deliverables introduced with the privacy-first restructure.

- `ropa.md`: record of processing activities
- `tom.md`: technical and organisational measures
- `vendor-register.md`: current processors and required replacement placeholders
- `retention-matrix.md`: retention and deletion baseline
- `incident-runbook.md`: breach and incident response path
- `dpia-checklist.md`: DPIA starter for the community feature
- `dpo-assessment.md`: DPO appointment assessment baseline

All placeholder operator details must be replaced before production.
